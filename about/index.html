关于漫游笔记

漫游笔记主要收录了我在工作中使用到的各种编程技巧，为了防止遗忘，我将它们详细记录下来。我采用了 Markdown 格式进行记录，并使用 Hugo 将它们转化为 HTML 格式，托管在 Github 上。在进行记录的时候，我使用 Obsidian 作为工具，受到卢曼卡片盒笔记法的启发，以此作为笔记的指导思想。
