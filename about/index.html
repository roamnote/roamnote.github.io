关于漫游笔记

我平时会使用 markdown 记笔记，后来想把这些笔记放到网上，我就使用 hugo 把这些 markdown 转化为了 html，并放到了公开的 web 服务器上。又因为注册到了 roamnote.com 这个域名，所以网站名直译为漫游笔记。

漫游笔记中的绝大多数笔记，都是我工作中需要用到的编程技巧，我担心遗忘，所以记录了下来。

我使用 obsidian 作为工具，卢曼卡片盒笔记法作为指导思想，来进行记录。实践步骤：使用 obsidian 实践卢曼卡片盒笔记法。
