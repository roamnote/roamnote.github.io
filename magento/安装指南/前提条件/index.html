前提条件

2024年04月07日

文档： https://experienceleague.adobe.com/en/docs/commerce-operations/installation-guide/prerequisites/overview

  • 首先要参考系统要求，设置一台主机，其中包括：
    • 安装 php
    • 安装 mysql
    • 安装 Search engine，OpenSearch 或者 Elasticsearch
  • 设置正确的文件权限
    • 读写权限：vendor、app/etc、pub/static、var、generated 和静态资源
      • 找到文件，给所有者增加文件的写入权限： find var generated vendor pub/static pub/media app/etc -type f -exec chmod u+w {} +
      • 找到目录以及所有子目录，给所有者增加文件的写入权限：find var generated vendor pub/static pub/media app/etc -type d -exec chmod u+w {} +
      • 给 bin/magento 文件的所有者添加执行权限：chmod u+x bin/magento
    • 想要了解 chmod 命令，可以参考：在 linux 中 如何使用 chmod 管理文件权限

如果运行 web 服务器的用户和管理 magento 项目的用户不是同一个，需要把后者加入到前者的组(或附加组)中，具体参考 https://experienceleague.adobe.com/en/docs/commerce-operations/installation-guide/prerequisites/file-system/configure-permissions#about-the-shared-group
