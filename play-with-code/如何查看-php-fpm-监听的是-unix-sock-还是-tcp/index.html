如何查看 php fpm 监听的是 UNIX sock 还是 TCP

2024年02月09日
  • 使用命令 find / -name www.conf 找到配置文件，例如： /etc/php-fpm.d/www.conf
  • 打开文件找到 listen = 127.0.0.1:9000 或者 listen = cat /var/run/php-fpm/php-fpm.sock，即可确认使用的是哪种方式
  • 更改为需要的方式，执行 systemctl restart php-fpm 重启 php-fpm
