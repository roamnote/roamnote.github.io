esxi 中的 linux 虚拟机硬盘扩容

2024年03月26日

注意：
以下操作可能会丢失数据，应该先备份数据。

  1. 把虚拟机关机
  2. 在 esxi 中增加硬盘容量
  3. 把虚拟机开机
  4. 使用 fdisk -l 查看硬盘大小

    [OCR] root@debian ~ (0.063s)[OCR] fdisk -1[OCR] Disk /devysda: 64 618, es719476736 bytes, 134217728 sectors[OCR] Disk model: ViFtuat atsk[OCR] Units: sectors of 1 * 512 = 512 bytes[OCR] Sector size (logical/physical): 512 bytes / 512 bytes[OCR] I/0 size (minimum/optimal): 512 bytes / 512 bytes[OCR] Disklabel type: dos[OCR] Disk identifier: 0xfe21000f[OCR] Device Boot Start End Sectors Size Id Type[OCR] /dev/sdal 2048 33554431 33552384 16G 83 Linux

  5. 执行 fdisk /dev/sda，进行操作，使用 p 查看分区详情、使用 d 删除当前分区、使用 n 新建分区、使用 w 保存分区设置。具体过程如下：
